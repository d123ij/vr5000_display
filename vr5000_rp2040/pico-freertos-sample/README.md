# Raspberry Pi Pico Sample application with FreeRTOS

## Introduction

- FreeRTOS kernel library for Pico SDK


## How to make

Pico SDK environment is required.

- Clone this repository with "--recursive" option.
- Type following commands.
    ```
    $ mkdir build
    $ cd build
    $ cmake ..
    $ make
    ```
